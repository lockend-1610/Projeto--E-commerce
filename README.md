# Autohub

Autohub é um sistema simples de e-commerce para oficina mecânica, desenvolvido para registrar compras e organizar o fluxo de pedidos de forma prática.

## Escopo do projeto

Este projeto tem como objetivo criar uma aplicação básica para uma oficina mecânica, permitindo que o cliente visualize produtos ou serviços, adicione itens ao carrinho e finalize uma compra.

### Funcionalidades principais
- Registro de compras.
- Carrinho de compras.
- Navegação entre 5 telas.
- Fluxo simples e sequencial de uso.
- Armazenamento dos dados em MySQL.

### Telas do sistema
1. Tela inicial.
2. Tela de produtos ou serviços.
3. Tela do carrinho.
4. Tela de confirmação da compra.
5. Tela final de sucesso.

### Sequência de navegação
**Tela inicial → Produtos/Serviços → Carrinho → Confirmação da compra → Sucesso**.

## Objetivo
O objetivo do sistema é simular um e-commerce simples para oficina mecânica, com foco no registro de compras e na organização do processo de finalização do pedido.

## Regras do sistema
- O usuário pode adicionar itens ao carrinho.
- O usuário pode revisar os itens antes de confirmar a compra.
- A compra só é registrada após a confirmação final.
- O sistema deve seguir uma sequência de telas definida.

## Observações
> Este projeto está em desenvolvimento e o escopo pode ser ajustado conforme a evolução do trabalho.
